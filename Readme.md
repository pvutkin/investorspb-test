Платформа для поиска стартапов и инвесторов — MVP

📋 Описание проекта

Минимальная версия SaaS-платформы для эффективного соединения стартапов, ищущих финансирование, с инвесторами (бизнес-ангелами, фондами). Ключевые особенности — система верификации, встроенный мессенджер и механизмы безопасности для построения доверия.

🎯 Ключевые роли пользователей

👥 Стартапы
- Создание детального профиля проекта
- Поиск подходящих инвесторов
- Безопасное взаимодействие
- Привлечение финансирования

💼 Инвесторы
- Создание профиля с критериями инвестирования
- Поиск перспективных проектов по сложным фильтрам
- Безопасное взаимодействие с основателями
- Заключение сделок

🚀 Основной функционал (MVP)

📊 Система профилей
**Общие обязательные поля (бесплатный тариф):**
- Базовая информация
- Контактные данные
- Сфера деятельности

## Стартапы: ##
- Название, описание (pitch)
- Стадия развития, требуемая сумма
- Ниша, год основания, сайт
- *Опционально:* команда, финансы, Pitch Deck

## Инвесторы: ##
- ФИО/название фонда, тип инвестора
- Предпочитаемые ниши и стадии
- Размер инвестиционного чека
- *Опционально:* портфолио, фото, опыт

🔍 Поиск и каталоги
- Общедоступные каталоги стартапов и инвесторов
- Расширенный поиск (доступ по подписке) с фильтрами:
  - Для стартапов: сфера, стадия, сумма, география, наличие MVP
  - Для инвесторов: тип, сфера, размер чека, стадия, география

💬 Система взаимодействия
- Внутренний мессенджер (без раскрытия прямых контактов)
- Отправка сообщений при активной подписке
- Уведомления на email и через Telegram-бота

🛡️ Модерация и безопасность
- Верификация: базовая (email, телефон) + расширенная (API Госуслуг)
- Модерация стартапов: авто — для сумм < 100k ₽, ручная — для сумм > 100k ₽
- Система жалоб: категории "мошенничество", "спам", "неадекват"
- Рейтинговая система для инвесторов

💰 Монетизация
- Многоуровневая SaaS-подписка (Basic, Pro)
- Платное размещение ("поднять в топ")
- Комиссия со сделок (требует дополнительной проработки)
- Интеграция с Яндекс.Пэй и Т-Банк

📡 API Endpoints

### 🔐 Аутентификация
- `POST /api/users/register/` — Регистрация пользователя
- `POST /api/users/login/` — Вход в систему
- `POST /api/users/logout/` — Выход из системы
- `POST /api/users/verify/` — Верификация аккаунта

### 👥 Пользователи
- `GET /api/users/profile/` — Получить профиль текущего пользователя
- `PUT /api/users/profile/` — Обновить профиль
- `GET /api/users/{id}/` — Получить публичный профиль пользователя

### 🚀 Стартапы
- `GET /api/startups/` — Список стартапов (с фильтрацией)
- `POST /api/startups/` — Создать стартап (только для пользователей с ролью "стартап")
- `GET /api/startups/{id}/` — Детальная информация о стартапе
- `PUT /api/startups/{id}/` — Обновить стартап (только владелец)
- `DELETE /api/startups/{id}/` — Удалить стартап (только владелец)

### 💼 Инвесторы
- `GET /api/investors/` — Список инвесторов (с фильтрацией)
- `POST /api/investors/` — Создать профиль инвестора (только для пользователей с ролью "инвестор")
- `GET /api/investors/{id}/` — Детальная информация об инвесторе
- `PUT /api/investors/{id}/` — Обновить профиль инвестора (только владелец)
- `DELETE /api/investors/{id}/` — Удалить профиль инвестора (только владелец)

### 💬 Мессенджер
- `GET /api/messaging/conversations/` — Список диалогов
- `GET /api/messaging/conversations/{id}/messages/` — Сообщения в диалоге
- `POST /api/messaging/conversations/{id}/messages/` — Отправить сообщение
- `POST /api/messaging/conversations/` — Начать новый диалог

### 🛡️ Модерация
- `POST /api/moderation/complaints/` — Создать жалобу
- `GET /api/moderation/complaints/` — Список жалоб (только модераторы)
- `PUT /api/moderation/complaints/{id}/` — Обновить статус жалобы (только модераторы)

### 💰 Платежи и подписки
- `GET /api/payments/subscriptions/plans/` — Доступные тарифные планы
- `POST /api/payments/subscriptions/` — Активировать подписку
- `GET /api/payments/subscriptions/current/` — Текущая подписка пользователя
- `POST /api/payments/process/` — Обработка платежа (интеграция с платежными системами)
